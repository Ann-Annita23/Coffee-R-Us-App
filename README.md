# ☕ Coffee R Us - A Coffee Inventory Management System
Welcome to Coffee R Us, a full-stack React  Single-Page Application designed to help coffee shop owners manage their inventory with ease.
 This project allows users to add new coffee products, view the current inventory, and perform update or delete operations on existing items. The application is built using React for the frontend and Node.js with Express for the backend, utilizing a simple in-memory data store for demonstration purposes.

 ## Key Architectual Features
 * **Dynamic Client-Side Routing** Implementing seamleass page transitions without full page browser reloads using `react-router-dom`.
 * **Custom React Hook**
 The use of useFetch custom hook to handle data fetching logic, promoting code reusability and separation of concerns.
 * **Proficient Modern Hook Implementation**
 Showcases effective usage of core React APIs including asynchronous data fetching, state management with useState, and side effects with useEffect.
 * ** Full CRUD Functionality**
 * **CREATE(POST)**: Add new coffee products to the inventory.
    * **READ(GET)**: View the list of all coffee products in the inventory.
    * **UPDATE(PUT)**: Edit existing coffee product details.
    * **DELETE(DELETE)**: Remove coffee products from the inventory.
    * **User-Friendly Interface**
    The application features a clean and intuitive user interface, making it easy for coffee shop owners to navigate and manage their inventory efficiently.
    * **lIVE Catalog Decoupled Filtering**
    When the user searches for a coffee products there is a live search feature that filters the inventory list in real-time as the user types, providing an efficient way to find specific products quickly.
    * **Responsive Design**
    Fully styled using **Bootstrap 5 CSS** layout systems, component cards, tables and functional utility tags for an appealling and responsive design that works well on various screen sizes and devices.
